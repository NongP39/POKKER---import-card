# Pokker — Card Import Tools

เครื่องมือสำหรับสร้างการ์ดกำหนดเองและนำเข้าเกม **Pokker** (Party Card Game for Android)

**เวอร์ชัน: v1.1.0 (Format v2)**

---

## ไฟล์ในโปรเจกต์นี้

| ไฟล์ | คำอธิบาย |
|------|-----------|
| `Pokker Card Encoder.exe` | โปรแกรม Windows แบบ standalone ไม่ต้องติดตั้งอะไรเพิ่ม |
| `encoder.html` | เครื่องมือเวอร์ชัน Browser ใช้งานได้ทุกอุปกรณ์ |
| `ตัวอย่างการ์ด.xlsx` | ไฟล์ Excel ตัวอย่างสำหรับดูรูปแบบการกรอกข้อมูล |
| `pokker_encoder.py` | Source code Python (สำหรับนักพัฒนา) |

---

## วิธีใช้งาน — Windows (.exe)

> **ไม่ต้องติดตั้ง Python หรือโปรแกรมใดๆ ทั้งสิ้น**

1. ดาวน์โหลดไฟล์ **`Pokker Card Encoder.exe`**
2. ดับเบิลคลิกเพื่อเปิดโปรแกรม
3. เลือกไฟล์ `.xlsx` หรือ `.json` โดย:
   - คลิก **"เลือกไฟล์"** หรือ
   - **ลากไฟล์แล้ววาง** (Drag & Drop) ลงในหน้าต่างโปรแกรมได้เลย
4. ตั้งชื่อชุดการ์ด (รองรับภาษาไทยสูงสุด ~21 ตัวอักษร) และชื่อผู้สร้าง (ถ้ามี)
5. กด **"สร้างภาพ PNG"**
6. ภาพจะถูกบันทึกที่ `Desktop/Pokker Export` โดยอัตโนมัติ

---

## วิธีใช้งาน — Browser (encoder.html)

> ใช้ได้บนทุกอุปกรณ์ที่มี Browser ไม่ต้องต่ออินเทอร์เน็ต

1. เปิดไฟล์ **`encoder.html`** ในเบราว์เซอร์ (Chrome / Edge / Firefox)
2. อัปโหลดไฟล์ Excel หรือ JSON
3. กรอกชื่อชุดการ์ดและผู้สร้าง
4. กด **"สร้างภาพ PNG"** แล้วดาวน์โหลด

## รูปแบบไฟล์ Excel

ดูไฟล์ตัวอย่าง **`ตัวอย่างการ์ด.xlsx`** — แถวแรกต้องเป็น header ดังนี้:

| หมวดหมู่ | เป้าหมาย | เนื้อหา | โทษ |
|----------|----------|---------|-----|
| Question | ทุกคน | ถ้าต้องอยู่บนเกาะร้างคนเดียว... | ดื่ม 1 แก้ว |
| Command | คนที่นั่งซ้ายมือ | โทรหาแม่แล้วบอกรัก | ดื่ม 2 แก้ว |
| Deeptalk | คนที่อายุน้อยที่สุด | ความฝันที่ยังไม่ได้ทำ... | เล่าให้ฟัง |

### หมวดหมู่ที่รองรับ

| หมวดหมู่ | ความหมาย |
|----------|----------|
| `Question` | คำถาม |
| `Command` | คำสั่ง |
| `Deeptalk` | คำถามลึกๆ |
| `Drink` | การดื่ม |
| `Skinship` | สัมผัส |
| `Minigame` | เกมย่อย |
| `Secret` | ความลับ |
| `Special` | พิเศษ |
| `Oops` | ซวย / โดน |

> ใส่ชื่อหมวดหมู่อื่นที่ไม่ได้อยู่ในรายการก็ได้ — เกมจะสร้างปุ่มหมวดหมู่ใหม่ให้อัตโนมัติ

## ข้อมูลทางเทคนิค (Format v2)

| ฟิลด์ | ขนาด |
|-------|------|
| Magic bytes | `POKR` |
| Version | `2` |
| ชื่อชุดการ์ด (set_name) | 64 bytes (~21 ตัวอักษรภาษาไทย) |
| ชื่อผู้สร้าง (creator) | 32 bytes |

> Format v1 และ v2 ทำงานร่วมกันได้ (game decoder รองรับทั้งสองเวอร์ชัน)

## หมายเหตุ

- ภาพที่ Export จะถูก encode ข้อมูลการ์ดไว้ใน pixel — ห้ามบีบอัดหรือแก้ไขภาพ
- หากข้อมูลมีขนาดใหญ่ โปรแกรมจะสร้างหลายภาพ — ต้อง import **ทุกภาพพร้อมกัน**
- รองรับ Windows 10 / 11 (64-bit)
