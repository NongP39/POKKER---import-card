# 🃏 Pokker — Card Import Tools

เครื่องมือสำหรับสร้างการ์ดกำหนดเองและนำเข้าเกม **Pokker** (Party Card Game for Android)

---

## 📦 ไฟล์ในโปรเจกต์นี้

| ไฟล์ | คำอธิบาย |
|------|-----------|
| `Pokker Card Encoder.exe` | โปรแกรม Windows แบบ standalone ไม่ต้องติดตั้งอะไรเพิ่ม |
| `encoder.html` | เครื่องมือเวอร์ชัน Browser ใช้งานได้ทุกอุปกรณ์ |
| `ตัวอย่างการ์ด.xlsx` | ไฟล์ Excel ตัวอย่างสำหรับดูรูปแบบการกรอกข้อมูล |
| `pokker_encoder.py` | Source code Python (สำหรับนักพัฒนา) |

---

## 🖥️ วิธีใช้งาน — Windows (.exe)

> **ไม่ต้องติดตั้ง Python หรือโปรแกรมใดๆ ทั้งสิ้น**

1. ดาวน์โหลดไฟล์ **`Pokker Card Encoder.exe`**
2. ดับเบิลคลิกเพื่อเปิดโปรแกรม
3. คลิก **"เลือกไฟล์"** แล้วเลือกไฟล์ `.xlsx` หรือ `.json`
4. ตั้งชื่อชุดการ์ด และชื่อผู้สร้าง (ถ้ามี)
5. กด **"สร้างภาพ PNG"**
6. ภาพจะถูกบันทึกที่ `Desktop / Pokker Export` โดยอัตโนมัติ

---

## 🌐 วิธีใช้งาน — Browser (encoder.html)

> ใช้ได้บนทุกอุปกรณ์ที่มี Browser (ไม่ต้องต่ออินเทอร์เน็ต)

1. เปิดไฟล์ **`encoder.html`** ในเบราว์เซอร์ (Chrome / Edge / Firefox)
2. อัปโหลดไฟล์ Excel หรือ JSON
3. กรอกชื่อชุดการ์ดและผู้สร้าง
4. กด **"สร้างภาพ PNG"** แล้วดาวน์โหลด

---

## 📋 รูปแบบไฟล์ Excel

ดูไฟล์ตัวอย่าง **`ตัวอย่างการ์ด.xlsx`** แถวแรกต้องเป็น header ดังนี้:

| ลำดับ | หมวดหมู่ | เป้าหมาย | เนื้อหา | โทษ |
|-------|----------|----------|---------|-----|
| 1 | Question | ทุกคน | ถ้าต้องอยู่บนเกาะร้างคนเดียว... | ดื่ม 1 แก้ว |
| 2 | Command | คนที่นั่งซ้ายมือ | โทรหาแม่แล้วบอกรัก | ดื่ม 2 แก้ว |
| 3 | Deeptalk | คนที่อายุน้อยที่สุด | ความฝันที่ยังไม่ได้ทำ... | เล่าให้ฟัง |

### หมวดหมู่ที่รองรับ

| หมวดหมู่ | ความหมาย |
|----------|----------|
| `Question` | คำถาม |
| `Command` | คำสั่ง |
| `Deeptalk` | คำถามลึกๆ |
| `Drink` | การดื่ม |
| `Skinship` | สัมผัส |
| `Minigame` | เกมย่อย |
| `Secret` | ความลับ |
| `Special` | พิเศษ |

## 📄 รูปแบบไฟล์ JSON (ทางเลือก)

```json
[
  {
    "Category": "Question",
    "Target": "ทุกคน",
    "Content": "ถ้าต้องอยู่บนเกาะร้างคนเดียว คุณจะเอาอะไรไปด้วย 3 อย่าง?",
    "Penalty": "ดื่ม 1 แก้ว"
  }
]
```

## ⚙️ สำหรับนักพัฒนา (Python)

```bash
pip install Pillow openpyxl
python pokker_encoder.py
```

**Build .exe ด้วยตัวเอง:**
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "Pokker Card Encoder" pokker_encoder.py
```

---

## 🎮 เกี่ยวกับ Pokker

**Pokker** คือเกมไพ่ปาร์ตี้บน Android ที่ให้ผู้เล่นสร้างชุดการ์ดกำหนดเองได้  
ฟีเจอร์นี้ช่วยให้คุณสร้างการ์ดจาก Excel แล้ว import เข้าเกมได้ง่ายๆ

---

## 📝 หมายเหตุ

- ภาพที่ Export จะถูก encode ข้อมูลการ์ดไว้ใน pixel — ห้ามบีบอัดหรือแก้ไขภาพ
- หากข้อมูลมีขนาดใหญ่ โปรแกรมจะสร้างหลายภาพ — ต้อง import **ทุกภาพพร้อมกัน**
- รองรับ Windows 10 / 11 (64-bit)
